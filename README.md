App de Gestión de Gastos del Hogar

Descripción del Proyecto
Esta aplicación permite a los usuarios llevar un control detallado de sus gastos domésticos y personales. El objetivo es centralizar la información financiera del hogar para facilitar el ahorro y la organización.

Objetivos:
- ¿Qué hace?: Permite registrar gastos, categorizarlos (comida, alquiler, ocio) y consultar el total.
- ¿Para qué sirve?: Para ayudar a las familias a saber en qué gastan su dinero y mejorar su economía.
- ¿A quién va dirigida?: A cualquier usuario que necesite organizar sus finanzas personales.

Tecnologías y Módulos:
- Bases de Datos: Diseño de tablas para usuarios, categorías y gastos.
- Programación: Lógica en Java y conexión mediante JDBC.
- Lenguajes de Marcas: Exportación de informes mensuales a formato XML.
- Sistemas Informáticos: Documentación del entorno de ejecución necesario.
