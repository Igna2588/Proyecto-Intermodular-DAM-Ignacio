Gestión de Gastos Familiares
Proyecto Intermodular de 1º DAM · Prometeo by The Power**  
Autor: Ignacio Arroyo

¿Qué es este proyecto?

Aplicación de escritorio desarrollada en Java que permite a una familia registrar, y controlar sus gastos e ingresos mensuales.

Muchas familias no saben con exactitud en qué gastan su dinero cada mes. Esta app centraliza toda esa información para facilitar el ahorro y la organización económica del hogar.

¿Qué problema resuelve?

Sustituye el uso de papel, por una herramienta pensada específicamente para el control del gasto familiar real: alquiler, suministros, suscripciones, ocio, transporte, etc.

Funcionalidades

- Registrar gastos e ingresos con fecha, importe y descripción
- Categorizar por tipo (vivienda, alimentación, transporte, ocio, suscripciones...)
- Asociar cada movimiento a un miembro de la familia
- Consultar el balance mensual
- Filtrar por categoría, miembro o mes
- Exportar datos a XML
